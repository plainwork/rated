# rated

Rated is a tiny menu bar app for quick 0 to 5 ratings. Name what you are rating, tap a score, and track it over time.

## Install

Install via Homebrew tap:

```bash
brew tap plainwork/rated https://github.com/plainwork/homebrew-rated
brew install --cask rated
```

Or install directly:

```bash
brew install --cask plainwork/rated/rated
```

## Use

- Click the Rated icon in the menu bar to open the rater.
- Type a name and tap the dots to rate.
- Each item allows one rating per day; the list shows the average.
- Toggle with the hotkey: control+option+command+R.
- Quit from the Quit button or press Cmd+Q.
